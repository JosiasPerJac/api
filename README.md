# PokéFinder: Explorando el mundo Pokémon

## ¿De qué trata este proyecto?
¡Hola! Este proyecto es mi aproximación a **FastAPI**. El objetivo es sencillo: crear una API que funcione como un puente entre nosotros y la famosa [PokéAPI](https://pokeapi.co/). Básicamente, quiero poder consultar datos de Pokémon de una forma más ágil, limpia y personalizada.

## ¿Por qué lo hice?
Más allá de aprender a usar FastAPI, el objetivo es entender cómo se conectan dos sistemas a través de la web. Quiero practicar cómo pedir información a un servidor externo, procesarla y presentarla de una forma que sea útil para cualquier persona que quiera consultar datos de sus Pokémon favoritos sin tener que lidiar con respuestas gigantes o complicadas.

## ¿Qué vas a encontrar aquí?
* **Simplicidad:** Un servicio directo para buscar datos puntuales.
* **Documentación al alcance:** Si entras a la ruta `/docs` cuando el servidor esté corriendo, verás una interfaz lista para probar los endpoints sin complicaciones.
* **Aprendizaje continuo:** Es el resultado de aplicar conceptos básicos de APIs REST en un entorno de desarrollo real.

## Tecnologías
Estoy trabajando con **Python** y el framework **FastAPI**, que hace que todo el proceso sea muy rápido y ligero. Para el servidor, uso **Uvicorn**, y para hablar con la PokéAPI, estoy manejando las peticiones de forma eficiente.
